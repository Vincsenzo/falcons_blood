# Falcon's Blood - Landing Page

A premium landing page for the historical strategy game "Falcon's Blood" (Magyarok Krónikái).

## Project Setup

This project was initialized manually as Node.js was not detected in the environment. To run the project, please ensure you have Node.js installed.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm (comes with Node.js)

### Installation

1.  Open a terminal in this directory.
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running Development Server

Start the development server:
```bash
npm run dev
```
Open your browser at `http://localhost:5173` (or the URL shown in the terminal).

### Building for Production

Build the project for production:
```bash
npm run build
```

## Features

- **Hero Section**: Cinematic introduction with generated concept art.
- **About Section**: Game features and historical context.
- **Gallery**: In-game screenshots (generated).
- **Pricing**: Three tiers (Standard, Deluxe, Collector's).
- **Responsive Design**: Fully responsive for mobile and desktop.
- **Modern Tech Stack**: React, Vite, Tailwind CSS.
